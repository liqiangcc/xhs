中兴软件开发一面

- 你开发的高并发社交点评平台是独立完成还是与他人合作的？该项目是导师安排还是自己找的练习项目？开发过程中是否有专门的网站或公司进行安排？
- 项目中的高并发是如何实现的？支持的并发数是多少？使用了哪些技术来应对高并发情况？具体如何操作？
- 在订单秒杀系统这个高并发典型场景中，如何实现库存预扣、提高效率？
- 项目使用的后台数据库是 MySQL，是否涉及存储过程的使用？
- 项目中表的数据量有多大？为提高查询效率采取了哪些优化手段？
- 在 Java 开发中，如何创建线程？
- 开发该项目时，是否有输出类似说明文档的资料？项目开发流程是怎样的（如需求分析、方案设计、开发过程等）？
- 项目实施过程中遇到了哪些比较难解决的问题？如何解决的？
- 除 Java 外，还掌握哪些语言？使用 Java 开发的时长是多久？使用 Python 的时长是多久？
- 在深度学习项目中，是否涉及大模型相关内容？具体情况如何？
- 平时有哪些爱好？作为班干部，在组织管理工作方面有哪些经验和心得？
