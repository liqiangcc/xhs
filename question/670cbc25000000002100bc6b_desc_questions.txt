1. 代码运行结果判断：
for (int i = 0; i < 3; i++) {
Thread t = new Thread(() -> {
System.out.println(i);
});
t.start();
}
面试官认为正确结果应当是输出了三个3，因为在绝大多数情况下，三个线程会在非常接近的时间间隔内启动，此时指向的i的值为3.
然而实际上编译有可能不通过，因为Lambda表达式内部的变量会被修改。
2. Linux指令：
有一文件为广告曝光日志，提取出其中的有效部分，即ad_id和user_id。利用正则表达式匹配并处理即可。
3. 二面算法：
寻找一棵二叉树中两个节点的最近公共节点。每个节点存储着指向其父节点的指针，要寻找给定两个节点的最近的公共父节点。
