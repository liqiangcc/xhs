# 🎯 八股文_Concept 分析框架

> 适用：知识点记忆、概念辨析、定义阐述
> 典型题干：「XXX 是什么」「XXX 有哪些」「XXX 和 YYY 的区别」

## 输出结构

```markdown
## 🎯 [原始题目]

### 1. 一句话定义
> 用一句话概括核心定义，抓住本质。

### 2. 为什么需要它？（解决了什么问题）
- 它出现之前的痛点是什么
- 它的设计初衷和解决的核心矛盾

### 3. 核心知识点拆解
| 维度 | 要点 | 关键细节 |
|------|------|----------|

### 4. 横向对比（如有近似概念）
| 对比项 | A | B |
|--------|---|---|
| 适用场景 | | |
| 优势 | | |
| 劣势 | | |

### 5. 面试官追问预判 🔮
- 追问 1：...
- 追问 2：...
- 追问 3：...

### 6. 记忆锚点 🧠
> 用一个类比或口诀帮助记忆核心要点。
```

## 分析要求

- **数据化**：能给数字的给数字（如 "HashMap 默认容量 16，负载因子 0.75"）
- **结构化**：用表格对比而非大段文字
- **追问深度**：预判追问要比原题深一个层次（L1 → L2）
