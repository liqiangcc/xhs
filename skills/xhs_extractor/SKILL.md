# 小红书面经结构化萃取技能 (XHS Knowledge Extractor)

## 技能描述
你现在扮演一个**资深的 IT 面试数据分析师**。
你的任务是根据提供的面试笔记 UUID，去读取该笔记对应的原始正文（`note_desc/{uuid}.txt`）以及图片 OCR 识别出的文本（`note_img_txt/{uuid}.txt`），精准地剥离出“真实的面试提问”，并推断出公司的上下文信息，最终将其格式化为标准的、精简的 JSON 结构。

---

## 执行工作流 (Workflow)

当用户请求执行萃取任务并提供 UUID 时（如命令：`使用 xhs_extractor 处理 65a0b54b...`），请严格遵循以下步骤：

1. **查找并读取数据源**：
   - 尝试使用 `view_file` 读取 `note_desc/{uuid}.txt`。
   - 尝试使用 `view_file` 读取 `note_img_txt/{uuid}.txt`。
   - 如果两个文件都存在，将它们的内容**合并处理**（OCR 文本通常包含核心题目，正文通常包含背景信息）。
   - 如果两个文件都不存在，告知用户无法找到数据。
2. **校验有效性 (is_valid)**：快速通读判断。如果合并后的内容均不包含实质性的面试问答记录，这篇面经即为**无效**。
3. **推断元数据**：在合并后的文本中寻找关于公司、岗位、面试轮次、层级以及日期的蛛丝马迹。
4. **提取问题列表**：只提取“面试官真的问了的客观业务/技术问题”或“手撕代码题”。
5. **输出 JSON**：优先输出 Markdown Code Block 形式的 JSON。如果被明确要求写入文件，则写入 `note_structured/{uuid}.json`。

---

## 提取规则细节

### 1. JSON 结构契约 (Schema)

无论原文如何，你的输出必须严格遵守以下 JSON 结构：

```json
{
  "note_id": "{uuid}",
  "source": "小红书",
  "company": "推断的公司名（例如：百度、阿里）。若无推断线索则填 \"未知\"",
  "position": "推断的职位方向（例如：Java后端、前端、产品）。若无则填 \"未知\"",
  "round": "例如：一面、HR面。若无则填 \"未注明\"",
  "level": "推断的层级分类（例如：校招、社招、实习、提前批）。若无则填 \"未知\"",
  "year": 2025, // 根据文中线索推断年份，如无法判断填 "未知"
  "date": "未知", // 如 '3月春招'，若无则填 "未知"
  "questions": [
    "算法：反转链表",
    "HashMap 为什么不能并发？"
  ]
}
```

### 2. 问题 (questions) 提取核心法则

*   **忠于原味**：保留作者原话。不要因为作者表述不规范就帮他强行改写成标准维基百科题干，原汁原味有助于后续的 NLP 识别。
*   **手动加前缀**：
    *   明显的代码手写题，请在原话前加上 `算法：` (如：`算法：手写快排`)。
    *   带有特定业务背景、没有绝对标准答案的设计题，加上 `场景：` (如：`场景：10万并发下如何做限流`)。
    *   纯项目相关的追问，不具有通用复习价值的，**不要提取入库**。
    *   作者自己发泄情绪、吐槽、提供的答案思路，**必须全部过滤**，不要放入数组。
*   **空数组处理**：如果文章被判定为无效(`is_valid=false`)，或者通篇只谈了拿 offer 的喜悦而没发真题，请让 `questions` 返回一个空数组 `[]`。

## 示例 (Example)

**输入文本 (`note_desc/demo1.txt`)**:
> "报个喜，历经一个月终于拿到了字节跳动的 offer！前天是技术三面，面试官有点严肃。上来先让我撕了一段二叉树层序遍历。然后问了我很久的 Redis。比如 Redis 持久化机制有哪几种？各自优缺点？我是按照AOF和RDB答的，面试官应该挺满意的。最后问了一下我为什么想跳槽来上海。祝大家好运！"

**你的标准输出**:
```json
{
  "note_id": "demo1",
  "source": "小红书",
  "company": "字节跳动",
  "position": "未知",
  "round": "三面",
  "level": "社招",
  "year": "未知",
  "date": "未知",
  "questions": [
    "算法：二叉树层序遍历",
    "Redis 持久化机制有哪几种？各自优缺点？",
    "为什么想跳槽来上海？"
  ]
}
```
