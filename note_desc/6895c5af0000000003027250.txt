Java 基础 三连问：
异常体系（Checked vs Unchecked 区别，实战中怎么选？）
泛型原理（类型擦除坑点，为什么不能实例化 T []？）
反射机制（应用场景 + 性能影响，怎么优化反射调用？）
集合 三巨头：
HashMap 底层
ConcurrentHashMap（JDK1.7 分段锁 vs 1.8 CAS+synchronized 区别）
ArrayList vs LinkedList
数据库 三核心：
索引类型（B + 树 vs 哈希索引，适用场景对比）
事务 ACID（隔离级别怎么实现？幻读如何解决？）
MVCC 原理（undo 日志 + Read View，快照读 vs 当前读区别）
	
💡 Step 2：第 4-5 天 - “场景模板” 硬套，答案结构化！(万能框架直接用！)
分布式场景 (e.g., 分布式事务)：
“方案选型 (TCC/SAGA/ 本地消息表) → 幂等设计 (唯一 ID 防重复) → 补偿机制 (定时任务 + 状态机) → 监控告警 (链路追踪 + 日志埋点)”
缓存问题 (e.g., 缓存异常)：
“穿透防护 (布隆过滤器 + 空值缓存) → 击穿解决 (互斥锁 + 热点永不过期) → 雪崩预防 (随机过期 + 多级缓存) → 一致性保障 (更新策略 + 定时对账)”
高并发接口 (e.g., 秒杀接口)：
“前端限流 (按钮置灰 + 验证码) → 网关拦截 (令牌桶 + IP 限制) → 服务抗压 (异步处理 + 队列削峰) → 兜底降级 (静态页面 + 熔断开关)”
